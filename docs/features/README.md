# Продуктовые функции

## Overview
Реестр описаний ключевых продуктовых функций для синхронизации команды.

## Problem Statement
Какие проблемы решают документы в этом каталоге:
- Потеря контекста о назначении функций
- Рассинхронизация команды относительно ценности продуктовых функций

## Solution Statement
Фиксация описаний значимых продуктовых функций в едином формате.

## Value Statement
- **Контекст:** Сохранение знаний о том, какие функции есть и зачем.
- **Синхронизация:** Единое понимание ценности продукта всей командой.
- **Повышение качества:** Процесс документирования способствует более четкой формулировке продуктовых функций.

## Architecture
Markdown-файлы в текущей директории.
