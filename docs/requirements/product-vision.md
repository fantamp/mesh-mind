# Продуктовое Видение: Mesh Mind

## Проблема

Компании и команды тонут в информации из разных источников:
- Email переписки с клиентами
- Сообщения в чатах (Telegram, Slack)
- Документы и регламенты
- Голосовые заметки

**Боль**: поддержка тратит **часы на поиск** информации в старых письмах и чатах. Нет единой базы знаний.

## Решение

**Mesh Mind** — интеллектуальный ассистент, который автоматически запоминает всю информацию и помогает быстро находить ответы.

### Основная Ценность

1. **Умное Запоминание** — автоматически сохраняет всю информацию из:
   - Telegram чатов (текст + голосовые сообщения)
   - Email писем (.eml файлы)
   - Документов (PDF, DOCX, TXT)
   
   Работает подобно Notebook LM от Google.

2. **Саммаризация** — создает краткие дайджесты:
   - По последним N сообщениям
   - За определенный период времени
   - По конкретной теме
   
   **Пример**: `/summary 20` → получаешь структурированное саммари последних 20 сообщений.

3. **Вопросы и Ответы** — отвечает на вопросы на основе базы знаний:
   - Поиск информации в письмах и чатах
   - Ссылки на источники
   - **Честно говорит "я не знаю"**, если ответа нет в БЗ
   
   **Пример**: "Какая у нас политика возврата?" → находит ответ в документах и указывает источник.

## Целевые Пользователи

- **Customer Support** — быстрые ответы клиентам на основе истории переписок
- **Команды разработки** — поиск решений в истории обсуждений
- **Малый бизнес** — работа с клиентскими запросами без найма больш

ой поддержки

## Сценарии Использования

### Сценарий 1: Поддержка Клиента

**Ситуация**: Клиент спрашивает про условия возврата.

**Традиционный способ**: менеджер ищет в email, папках, документах → 15-30 минут.

**С Mesh Mind**:
1. Менеджер спрашивает бота: "Какая политика возврата?"
2. Бот находит информацию в БЗ за 5 секунд
3. Возвращает ответ с источником

**Результат**: ⚡ время поиска **сокращено с 30 минут до 30 секунд**.

### Сценарий 2: Дайджест Переписки

**Ситуация**: Нужно быстро понять, о чём говорили в чате за последние 2 дня.

**С Mesh Mind**:
1. Команда `/summary 2 дня` в Telegram
2. Бот создает структурированное саммари:
   - Ключевые темы
   - Принятые решения
   - Action items

**Результат**: экономия времени на чтение всей истории.

### Сценарий 3: База Знаний из Email

**Ситуация**: Компания получила 5000 писем от клиентов за год.

**С Mesh Mind**:
1. Загрузка .eml файлов через CLI
2. Автоматическая индексация в БЗ
3. Мгновенный поиск по любому вопросу

**Результат**: вся история переписок доступна для поиска.

## Уникальность

1. **Мультиагентная архитектура** — специализированные агенты для разных задач (не один "универсальный" бот)
2. **Поддержка 3 языков** — украинский, русский, английский (для голосовых)
3. **Честность** — бот не придумывает ответы, говорит "я не знаю"
4. **Изоляция по чатам** — каждый Telegram чат видит только свои данные
5. **Telegram интеграция** — работает прямо в привычном мессенджере

## Метрики Успеха

- **Время поиска информации**: с 2+ часов → до 30 секунд
- **Точность ответов**: QA agent должен правильно отвечать в 80%+ случаев
- **"Я не знаю" rate**: 10-15% (честно признается, когда нет ответа)

## Roadmap

### MVP (текущая стадия)
- ✅ Telegram bot с базовыми командами
- ✅ QA Agent (вопросы-ответы)
- ✅ Chat Summarizer (суммаризация)
- ✅ Загрузка документов и email

### Следующие шаги
- Улучшение базы знаний (UI для редактирования)
- Кастомная суммаризация по инструкциям
- Тегирование документов
- Confidence score для ответов
