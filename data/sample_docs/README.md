# Sample Documents

This directory contains sample files used for testing the CLI ingestion tool.

## Files

- **sample.txt** - Plain text file example
- **sample.md** - Markdown file example  
- **sample.eml** - Email (.eml) file example

## Usage

These files are used by `cli/main.py` for:
- Testing bulk ingestion functionality
- Validating different file format parsers
- CLI development and debugging

**Do not delete** - these files are part of the testing infrastructure.
