# Обзор Проекта

## О Проекте

**Mesh Mind** — интеллектуальный ассистент на основе мультиагентной архитектуры (Google ADK).

**Возможности**: умное запоминание сообщений из Telegram, суммаризация, вопросы-ответы на базе знаний.
