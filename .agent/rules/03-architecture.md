# Архитектурные Принципы

## Границы Компонентов

Проект разделён на независимые компоненты:
- **Telegram Bot** (`telegram_bot/`) — приложение для работы с Telegram API
- **AI Core** (`ai_core/`) — центральный сервис с агентами и хранилищем

**Важно**: не нарушай границы между компонентами. Если нужно изменить архитектуру — сообщи об этом.

## Критичные Правила

### Изоляция по chat_id
**Обязательно**: все tools фильтруют данные по `chat_id`. Агенты видят только данные своего чата.

### Использование ADK
- **Использовать**: Google Agent Development Kit (ADK) для создания агентов
- Документация: https://google.github.io/adk-docs/
