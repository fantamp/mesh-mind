# Обзор Проекта Mesh Mind

## Краткое Описание

**Mesh Mind** — интеллектуальный ассистент для автоматической обработки сообщений и работы с базой знаний. Система построена на основе мультиагентной архитектуры с использованием Google Agent Development Kit (ADK).

### Основная Ценность

1. **Умное Запоминание**: автоматически сохраняет всю информацию из чатов, документов, email (подобно Notebook LM)
2. **Суммаризация**: создает краткие саммари по подмножеству сообщений
3. **Вопросы и Ответы**: отвечает на вопросы на основе базы знаний, честно говорит "я не знаю" при отсутствии информации

## Стек Технологий

- **Язык**: Python 3.10+
- **AI Framework**: Google Agent Development Kit (ADK)
- **Векторная БД**: ChromaDB
- **Реляционная БД**: SQLite
- **Bot Framework**: python-telegram-bot v20+
- **API**: FastAPI
- **UI**: Streamlit

## Архитектура

Система состоит из трех основных компонентов:

1. **Telegram Bot** — отдельное приложение для работы с Telegram API (Long Polling)
2. **AI Core** — центральный сервис с агентами, API и UI
3. **CLI Tools** — инструменты для массовой загрузки данных

Подробная архитектура описана в docs/architecture/.

## Ключевые Агенты

1. **Orchestrator** — главный координатор запросов
2. **Chat Summarizer** — суммаризация истории чата
3. **QA Agent** — ответы на вопросы из базы знаний
4. **Chat Observer** — поиск конкретных сообщений

Детальное описание в docs/requirements/multi-agent-system.md.
