# Спецификация: Мультимодальные Обработчики

## Цель
Создать эндпоинты для обработки Голоса и Изображений через API.

## Требования
1.  **API Эндпоинты:**
    *   `POST /transcribe`: Принимает аудиофайл (или путь). Вызывает Gemini Audio.
    *   `POST /analyze_image`: Принимает файл изображения (или путь). Вызывает Gemini Vision.
2.  **Интеграция:**
    *   Подключить к `src/services/gemini.py`.
    *   Возвращать текстовый результат.

## Верификация
*   [ ] Отправить тестовый .ogg файл на `/transcribe` (имитируя загрузку файла или путь).
*   [ ] Отправить тестовый .jpg файл на `/analyze_image`.
*   [ ] Убедиться, что JSON ответ содержит текстовое описание/транскрипцию.
