# Спецификация: RAG Агенты (Summarizer & QA)

## Цель
Реализовать основных AI агентов, используя паттерны Google ADK.

## Требования
1.  **Summarizer Agent:**
    *   Вход: Список сообщений или текст.
    *   Логика: Сформировать промпт для суммаризации контента.
    *   Выход: Текст саммари.
2.  **QA Agent:**
    *   Вход: Пользовательский запрос.
    *   Логика:
        *   Найти релевантные чанки в ChromaDB.
        *   Сформировать промпт с контекстом.
        *   Сгенерировать ответ, используя Gemini.
3.  **Интеграция:**
    *   Экспортировать их как функции в `src/services/agents.py`.

## Верификация
*   [ ] Вручную наполнить БД некоторыми фактами (например, "The project name is Mesh Mind").
*   [ ] Запустить QA Agent с запросом "What is the project name?".
*   [ ] Убедиться, что ответ корректен и основан на контексте.
