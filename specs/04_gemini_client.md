# Спецификация: Обертка Клиента Gemini

## Цель
Реализовать обертку вокруг Google ADK / Gemini API для централизованной обработки аутентификации и вызовов модели.

## Требования
1.  **Конфигурация:**
    *   Читать `GOOGLE_API_KEY` из переменных окружения.
2.  **Класс Клиента:**
    *   Создать `src/services/gemini.py`.
    *   Метод `generate_text(prompt: str) -> str`.
    *   Метод `generate_embedding(text: str) -> list[float]`.
    *   Метод `analyze_image(image_path: str, prompt: str) -> str`.
    *   Метод `transcribe_audio(audio_path: str) -> str`.
3.  **Обработка Ошибок:**
    *   Корректная обработка ошибок API (повторные попытки или понятные исключения).

*   [ ] Убедиться, что вывод не пустой.
