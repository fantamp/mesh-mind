# Спецификация: Сервис Ингестии (Ingestion Service)

## Цель
Логика для обработки входящего текста и файлов, их разбиения на чанки и сохранения в Vector DB.

## Требования
1.  **Логика Ингестии:**
    *   Создать `src/services/ingestion.py`.
    *   Функция `ingest_text(content: str, metadata: dict)`.
    *   Функция `ingest_file(file_path: str, metadata: dict)`.
2.  **Чанкинг (Chunking):**
    *   Реализовать простой разделитель текста (например, по параграфам или фиксированному размеру).
3.  **Взаимодействие с Vector Store:**
    *   Использовать клиент ChromaDB (из Спеки 02) для сохранения чанков и эмбеддингов.
    *   Убедиться, что метаданные (источник, дата) прикреплены к каждому чанку.

## Верификация
*   [ ] Запустить скрипт для ингестии тестового текстового файла.
*   [ ] Сделать запрос в ChromaDB, чтобы убедиться, что чанки существуют и имеют правильные метаданные.
