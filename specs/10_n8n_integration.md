# Спецификация: Интеграция с n8n

## Цель
Настроить воркфлоу n8n для оркестрации взаимодействий с Telegram и файловой системой.

## Требования
1.  **Python Скрипты для n8n:**
    *   Создать `scripts/n8n/router.py`: Анализ типа сообщения (text/voice/image).
    *   Создать `scripts/n8n/check_command.py`: Определение команд `/ask` или `/summary`.
2.  **Дизайн Воркфлоу (JSON):**
    *   Экспортировать структуру воркфлоу в `scripts/n8n/workflow_telegram.json` (как эталон или бекап).
    *   Воркфлоу должен вызывать API эндпоинты, определенные в Спеках 03 и 07.
3.  **Триггеры:**
    *   Telegram Trigger (Polling).
    *   File System Trigger (для `input_docs`).

## Верификация
*   [ ] Импортировать воркфлоу в n8n.
*   [ ] Отправить сообщение боту в Telegram.
*   [ ] Проверить логи выполнения в n8n, чтобы убедиться в правильной маршрутизации и вызове локального API.
