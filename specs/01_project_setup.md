# Спецификация: Настройка Проекта и Инфраструктуры

## Цель
Инициализировать окружение проекта, установить зависимости и настроить базовую структуру директорий.

## Требования
1.  **Python Окружение:**
    *   Создать виртуальное окружение `venv`.
    *   Установить необходимые пакеты: `fastapi`, `uvicorn`, `streamlit`, `chromadb`, `google-adk`, `python-dotenv`, `requests`.
    *   Создать `requirements.txt`.
2.  **Node.js Окружение:**
    *   Убедиться, что `n8n` установлен глобально или локально (предпочтительно глобально `npm install -g n8n`).
3.  **Конфигурация:**
    *   Создать `.env.example` с плейсхолдерами для:
        *   `GOOGLE_API_KEY`
        *   `TELEGRAM_BOT_TOKEN`
        *   `CHROMA_DB_PATH`

## Верификация
*   [ ] Запустить `source venv/bin/activate && pip list` для проверки зависимостей.
*   [ ] Запустить `n8n --version` для проверки n8n.
*   [ ] Проверить наличие всех директорий в `data/` и `src/`.
*   [ ] Проверить наличие файла `.env` (скопировать из примера).
