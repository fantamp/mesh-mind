# Спека 02: Удаление Web UI и CLI

## Цель
Удалить из проекта Web UI (Streamlit) и CLI инструменты, так как массовая загрузка данных больше не нужна.

## Удаление Web UI

### 1. Удалить файлы
- **Удалить:** `ai_core/ui/` (вся папка со всем содержимым)
- Это включает:
  - `ai_core/ui/main.py`
  - `ai_core/ui/pages/`
  - Все зависимости от Streamlit

### 2. Обновить зависимости
- **Удалить из `requirements.txt`:**
  - `streamlit`
  - Любые другие зависимости, используемые только UI

## Удаление CLI

### 3. Удалить файлы
- **Удалить:** `cli/` (вся папка)
- Это включает:
  - `cli/main.py`
  - Все CLI утилиты для массовой загрузки

### 4. Обновить зависимости
- **Удалить из `requirements.txt`:**
  - `typer`
  - `tqdm`
  - Любые другие зависимости, используемые только CLI

## Обновление Makefile

### 5. Удалить команды
- **Удалить:** `make run-ui` (или аналогичные команды для запуска Streamlit)
- **Удалить:** `make cli` (или аналогичные команды для CLI)
- **Оставить:** `make run-bot`, `make test`, `make install`

## Обновление тестов

### 6. Удалить тесты
- **Удалить:** `tests/test_ui_smoke.py`
- **Удалить:** `tests/test_cli.py`
- Удалить любые другие тесты, связанные с UI или CLI
