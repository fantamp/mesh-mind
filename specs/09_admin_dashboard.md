# Спецификация: Панель Администратора (Streamlit)

## Цель
UI для просмотра базы знаний и управления данными.

## Требования
1.  **Приложение Streamlit (`src/ui/admin.py`):**
    *   **Tab 1: Knowledge Base:** Список чанков из ChromaDB. Возможность поиска/фильтрации.
    *   **Tab 2: Manual Ingest:** Текстовое поле для вставки текста и кнопка "Ingest".
    *   **Tab 3: Chat:** Простой чат-интерфейс для тестирования QA Agent.
2.  **Запуск Скрипта:**
    *   Команда для запуска: `streamlit run src/ui/admin.py`.

## Верификация
*   [ ] Запустить приложение.
