# Спецификация: Базовый API Бэкенда

## Цель
Создать скелет приложения FastAPI, который будет служить основным провайдером логики.

## Требования
1.  **Приложение FastAPI:**
    *   Создать `src/api/main.py`.
    *   Настроить CORS middleware (разрешить всё для локальной разработки).
    *   Настроить логирование (stdout).
2.  **Health Check:**
    *   Эндпоинт `GET /health`, возвращающий `{"status": "ok"}`.
3.  **Запуск Сервера:**
    *   Скрипт или команда для запуска uvicorn: `uvicorn src.api.main:app --reload`.

## Верификация
*   [ ] Запустить сервер.
*   [ ] Выполнить `curl http://localhost:8000/health` и ожидать 200 OK.
